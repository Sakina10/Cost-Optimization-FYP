# Cost Optimization Analysis for Businesses

A Final Year Project focused on identifying cost inefficiencies, improving profitability, and supporting data-driven business decisions using exploratory analysis and machine learning.

## Overview

This project analyzes business transaction data to answer critical questions such as:

- Which factors contribute to profit loss?
- How do discounts impact profitability?
- Which products or segments are inefficient?
- How can businesses optimize operational costs?

The project combines:

- Data Cleaning and Validation  
- Exploratory Data Analysis (EDA)  
- Feature Engineering  
- Predictive Modeling  
- Business Insight Generation  

---

## Objectives

- Detect cost leakages in business operations  
- Analyze the relationship between sales, cost, discount, and profit  
- Build predictive models for profit estimation  
- Compare baseline and improved models  
- Recommend practical cost optimization strategies  

---

## Dataset

**Dataset Used:** Superstore Sales Dataset

Includes variables such as:

- Quantity

## Machine Learning Models

### Model 1 — Linear Regression
Baseline model used for initial profit prediction.

### Model 2 — Random Forest Regressor
Improved model using:
- Additional features
- Outlier handling
- Feature importance analysis
- Cross-validation

### Model Comparison
The second model significantly outperformed the baseline and produced stronger predictive performance.
